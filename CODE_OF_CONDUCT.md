# Contributor Covenant Code of Conduct

## Our pledge

We pledge to make participation in this project a harassment-free experience
for everyone, regardless of age, body size, disability, ethnicity, sex
characteristics, gender identity and expression, experience, education,
socioeconomic status, nationality, appearance, race, caste, color, religion,
or sexual identity and orientation.

## Our standards

Positive behavior includes empathy, constructive feedback, respect for
different viewpoints, and responsibility for mistakes. Unacceptable behavior
includes harassment, intimidation, personal attacks, sexualized attention,
publishing others' private information, or encouraging unauthorized security
testing.

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions
that violate this code. Report conduct concerns privately to the maintainers
through the repository's listed contact channel. Maintainers will respect the
privacy and safety of reporters.

This code is adapted from Contributor Covenant version 2.1.
